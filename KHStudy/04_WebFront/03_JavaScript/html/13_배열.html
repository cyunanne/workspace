배열(Array)

        변수를 묶음으로 다루는 것(변수가 연속적으로 나열됨)
        
        * js 변수의 특징 : 값이 저장되기 전 까지는 자료형을 알 수 없다 
            즉, 자료형이 지정되지 않는다
        
        js 배열의 특징
        1. 자료형 제한 X
        2. 길이 제한 X (자바의 Collection.List 처럼)

        js 배열 선언 방법
        1. const arr = new Array();  // 0칸 짜리 배열(비어있는 배열) 생성
           const arr = new Array(3); // 3칸 짜리 배열 생성
           const arr = [];           // 비어있는 배열 생성
           const arr = [1, 2, 3];    // 배열 생성 및 초기화

배열 관련 함수

        * 배열 관련 함수는 prototype 이 array 인 배열에만 사용이 가능하다
            cf. Object, NodeList, HTMLCollection 등은 상용 불가!

        (Stack 구조 관련 함수)
        배열명.push()    : 배열 마지막 요소로 추가
        배열명.pop()     : 배열 마지막 요소를 반환 & 삭제
        배열명.unshift() : 배열 첫번째 요소로 추가
        배열명.shift()   : 배열 첫번째 요소를 반환 & 삭제

        배열명.indexOf('값') : 배열 내 요소 중 값이 일치하는 요소의 index를 반환(단, 없으면 -1 반환)
            -> 중복 검사에 많이 사용
        배열명.sort([정렬기준함수]) : 배열 정렬
            cf. 정렬기준함수 default = **문자열** 오름차순
        배열명.toString() : 모든 요소를 하나의 문자열로 반환하며, 구분자로 ',' 사용
        배열명.join('구분자') : 구분자를 사용하여 모든 요소를 하나의 문자열로 반환
        배열명.forEach( function(value) {} ) : 배열 요소를 순차접근하여 function 매개변수로 전달 및 함수 실행
