DOM(Document Object Model)

        웹 문서(HTML)의 모든 요소를 객체 형식으로 표현한 것

DOM을 이용한 요소 접근

        1. id 로 접근하기 : document.getElementById("id 속성값");
            -> id 가 일치하는 요소(객체)를 얻어옴

        2. class 로 접근하기 : document.getElementsByClassName("class 속성값");
            -> class 가 일치하는 모든 요소를 얻어와 배열 형태로 반환
        
        3. name 으로 접근하기 : document.getElementsByName("name 속성값");
            -> name이 일치하는 모든 요소를 얻어와 배열(NodeList, 유사배열) 형태로 반환

        4. tag 로 접근하기 : document.getElementsByTagName("tag 속성값");
            -> tag가 일치하는 모든 요소를 얻어와 배열 형태로 반환

        5. CSS 선택자로 접근하기

            1) document.querySelector("CSS 선택자");
                -> 선택자에 해당하는 요소 중 첫 번째 요소 반환 
        
            2) document.querySelectorAll("CSS 선택자");
                -> 선택한 모든 요소를 배열(NodeList, 유사배열) 형태로 반환
