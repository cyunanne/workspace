ol 태그 : Ordered List, 순서가 있는 목록

        type 속성 : 순서를 나타내는 형식을 지정

        - 미작성 : 1 부터 1 씩 증가하는 숫자
        - type="a" : 영어 소문자 순서(a, b, c, ...)
        - type="A" : 영어 대문자 순서(A, B, C, ...)
        - type="i" : 로마 소문자 순서(i, ii, iii, ...)
        - type="I" : 로마 대문자 순서(I, II, III, ...)
        - start='4' : 몇 부터 시작?
        - reversed : 반대
  1. 리코타 치즈 샐러드
  2. 호박 샐러드
  3. 탭키 안되는딩
  4. 이미 화살표 써서 그런가
  5. 라면 맛잇겟다...
